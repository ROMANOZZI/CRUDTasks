# CRUD Tasks

 an interactive to do list With MERN Stack
## setup

to export the tasks DB

```bash
cd <the root directory>
mongorestore --db tasks dump/tasks

```
to setup the Back-end
```bash
cd Back-end
npm install
npm run dev

```
to setup the Front-end
```bash
cd Front-end
npm install
npm run dev


```
